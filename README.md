# AIUniExercises

Collection of exercises for Artificial Intelligence course

## Available exercises:
 
* [`E.A.5.2 CSP N-Queens`](E.A.5.2%20CSP%20N-Queens/): N-Queens problem modeled as a CSP and implemented in MiniZinc.

* [`E.A.5.3 CSP Edge Colouring`](E.A.5.3%20CSP%20Edge%20Colouring/): Edge Colouring problem modeled as a CSP and implemented in MiniZinc.

* [`E.A.5.4 Generalised Arc Consistency`](E.A.5.4%20Generalised%20Arc%20Consistency/): Exercise on node-consistency and arc-consistency

* [`E.A.5.1 Steepest Descent`](E.A.5.1%20Steepest%20Descent/): Exercise on steepest-descent local search algorithm

* [`E.A.5.5 CSP Backtracking`](E.A.5.5%20CSP%20Backtracking/): Exercise on backtracking

* [`E.A.5.6 CSP Cards`](E.A.5.6%20CSP%20Cards/): Cards problem modeled as a CSP and implemented in Minizinc

* [`E.A.5.7 CSP Cards 2`](E.A.5.7%20CSP%20Cards%202/): Cards 2 problem modeled as a CSP and implemented in Minizinc
